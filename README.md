# Employee Management System

A full-stack Employee Management System developed using Java Spring Boot, MySQL, HTML, CSS, Bootstrap, and JavaScript.

## Features

- User Login & Logout
- Add Employee
- View Employee Records
- Search Employees
- Edit Employee Details
- Delete Employees
- Responsive Dashboard UI

## Technologies Used

### Frontend
- HTML
- CSS
- Bootstrap
- JavaScript

### Backend
- Java Spring Boot
- REST API

### Database
- MySQL
